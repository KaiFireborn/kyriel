Kyriel Keyboard OS Remapping Guide
===================================

This file documents all key remappings needed for the Kyriel keyboard across different operating systems.
Use this as the source of truth; OS-specific config files (AHK, keyd, etc.) are derived from this.

ARROWS
======
AltGr + L -> ↑
AltGr + K -> ↓
AltGr + J -> ←
AltGr + ; -> →
AltGr + [ -> ⇐
AltGr + ] -> ⇒

UMLAUTS
=======
AltGr + U -> ü
AltGr + A -> ä
AltGr + O -> ö
AltGr + S -> ß
AltGr + C -> ā
AltGr + Shift + A -> Ä
AltGr + Shift + U -> Ü
AltGr + Shift + O -> Ö
AltGr + Shift + C -> Ā

NUMBER ROW + ALTGR
==================
AltGr + 1 -> ↵
AltGr + 2 -> €
AltGr + 3 -> °
AltGr + 4 -> Δ
AltGr + 5 -> ∞
AltGr + 6 -> ·
AltGr + 7 -> ✓
AltGr + 8 -> ✕
AltGr + 9 -> μ
AltGr + 0 -> ⌀

IMPORTANT PUNCTUATION
===================
AltGr + - -> −
AltGr + = -> ≠
AltGr + , -> ≤
AltGr + . -> ≥
AltGr + X -> ≈
SC056 -> §
Shift + \ -> ±

LOGIC SYMBOLS
=============
AltGr + N -> ¬
AltGr + / -> ⇔
AltGr + ' -> ∧
AltGr + \ -> ∨
AltGr + ` -> ↯
Shift + N -> ∨



BRIGHTNESS CONTROL
===================
F18 -> Shift+Alt+Up (center monitor brightness up)
F19 -> Shift+Alt+Down (center monitor brightness down)

NOTES
=====
- ALGR() in QMK = AltGr (Right Alt) on Windows/Linux
- These remaps are OS-agnostic at the character level
- Windows: Use AHK script (kyriel_keys.ahk)
- Linux: Use keyd config (kyriel_keys.conf)
- macOS: May require Karabiner-Elements or similar
